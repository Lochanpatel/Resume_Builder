# Resume Builder

A small resume builder project to create, preview, and export resumes using customizable templates.

## Features
- Create and edit resume data (personal info, experience, education, skills)
- Multiple templates and styles
- Live preview and template selection
- Export to PDF / print-friendly output

## Quick Start
1. Open this folder in VS Code.
2. Inspect the project to determine backend/frontend stacks and dependencies.
3. Install dependencies (examples):

```powershell
npm install
pip install -r requirements.txt
```

4. Run the dev server (examples):

```powershell
npm run dev
python -m flask run
```
